# Plonk

Plonk is a crappy and incomplete 3D engine written in C++ using Vulkan. Only a plonker would use this engine.

## How to build and run

* Boot into Linux
* git submodule update --init
* make run

## Dependencies

* CMake
* Vulkan
* GLFW
